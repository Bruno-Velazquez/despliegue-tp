Desarrollé un convertidor de números romanos usando JavaScript y HTML. El programa permite:
Convertir números enteros a números romanos.
Convertir números romanos a números enteros.

**EL PROYECTO INCLUYE:**

Una interfaz web (SPA) para mostrar y recibir los datos del usuario.

Una API en Node.js con Express para manejar las conversiones de forma dinámica.

Validaciones para asegurar que los números romanos y enteros ingresados sean correctos.

**FUNCION PRINCIPAL**

Para convertir un número a romano, llamás la ruta /a2r y le pasás el número como parámetro arabic.

Para convertir un número romano a normal, llamás la ruta /r2a y le pasás el número romano como parámetro roman.
